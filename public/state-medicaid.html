State Medicaid Directory

Medicaid is administered by each state within federal requirements, so program names, benefits, and eligibility differ from state to state.

Every state and the District of Columbia runs its own Medicaid program — often under a distinctive brand name (Medi-Cal, MassHealth, TennCare, SoonerCare, and so on). Use this directory to reach the official state agency for authoritative eligibility, enrollment, covered-benefit, and provider information.

Always confirm at the source. Program names and agency websites change. These links point to official state government sites; if a link has moved, search for the state's Department of Health, Human Services, or Health Care Authority. MediPrimer is not affiliated with any state agency.
